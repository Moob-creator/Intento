# 通知设置 - 使用指南

配置 Intento 的智能提醒系统，永不错过重要的任务截止日期。

## 🎯 功能概述

Intento 的通知系统提供：
- 桌面系统级通知
- 灵活的截止日期提醒
- 可自定义的提醒时间
- 测试功能验证设置

---

## 🚀 快速设置

### 访问通知设置

**方式 1: 键盘快捷键**
```
按 ⌘, (macOS) 或 Ctrl+, (Windows/Linux)
```

**方式 2: Command Palette**
```
1. 按 ⌘K
2. 输入"设置"或"Settings"
3. 按 Enter
```

**方式 3: 点击按钮**
```
点击顶部栏的设置图标（齿轮）
```

---

## ⚙️ 通知设置项

### 1. 主开关

**启用桌面通知**
- 开启/关闭所有桌面通知
- 关闭后，所有提醒和通知都会被禁用
- 建议保持开启以接收重要提醒

### 2. 截止日期提醒

**提醒我的截止日期**
- 在任务到期前发送提醒
- 只有开启主通知开关后才能配置
- 可以选择不同的提前时间

**提前提醒时间选项:**

| 选项 | 说明 | 适用场景 |
|------|------|---------|
| 15 分钟前 | 紧急任务 | 会议、电话等即时任务 |
| 30 分钟前 | 短期任务 | 需要快速准备的任务 |
| 1 小时前 | 常规任务 | 日常工作任务 |
| 2 小时前 | 重要任务 | 需要充足准备时间 |
| 1 天前 | 长期任务 | 项目截止、重大事件 |

**默认设置:** 1 小时前

### 3. 测试通知

**测试按钮功能:**
- 立即发送一个测试通知
- 验证通知系统是否正常工作
- 检查系统权限是否配置正确

**使用场景:**
- 首次配置通知设置后
- 更改操作系统通知权限后
- 怀疑通知不工作时

## 💡 使用技巧

### 推荐配置

**高频任务用户:**
```
✓ 启用桌面通知
✓ 提醒截止日期
  提前时间: 1 小时前
```

**需要充足准备时间:**
```
✓ 启用桌面通知
✓ 提醒截止日期
  提前时间: 1 天前
```

**会议和紧急事务:**
```
✓ 启用桌面通知
✓ 提醒截止日期
  提前时间: 15 分钟前
```

### 工作流程建议

1. **首次使用**
   - 开启所有通知
   - 设置提醒时间为 1 小时
   - 点击"测试通知"验证

2. **日常使用**
   - 为不同优先级任务设置不同截止时间
   - 高优先级任务可以设置更早的提醒
   - 定期检查通知是否正常工作

3. **专注时段**
   - 需要专注时可临时关闭通知
   - 完成工作后记得重新开启
   - 或使用系统的"勿扰模式"

## 🔔 通知行为说明

### 通知触发时机

- **自动检查**: 每小时自动检查即将到期的任务
- **提醒条件**: 任务截止时间在设置的提前时间内
- **重复提醒**: 同一任务只提醒一次

### 通知内容

**标准通知格式:**
```
标题: 任务截止提醒
内容: [任务标题] 将在 [时间] 到期
```

**测试通知格式:**
```
标题: 测试通知
内容: 通知功能正常！你将收到任务截止日期的提醒。
```

### 通知样式

- **系统原生**: 使用操作系统原生通知样式
- **自动消失**: 通知会在几秒后自动消失
- **可点击**: 点击通知可打开 Intento（未来功能）

## 🖥️ 平台支持

### macOS

**支持版本:** macOS 10.14+

**配置步骤:**
1. 首次使用时，系统会请求通知权限
2. 点击"允许"
3. 如果误点"拒绝"，需要手动开启：
   - 打开"系统设置" > "通知"
   - 找到 Intento
   - 开启"允许通知"

**⭐ 重要：设置通知样式**

为了确保任务提醒能弹出桌面横幅，建议：

1. 打开"系统设置" > "通知"
2. 找到"Intento"
3. 配置以下设置：
   - ☑ **允许通知** - 开启
   - ⦿ **通知样式** - 选择 **"提醒"** (Alerts) ⭐ 推荐
   - ☑ **播放提示音** - 开启
   - ☑ **在锁定屏幕上显示** - 开启
   - ☑ **在通知中心显示** - 开启

**通知样式说明:**
- **提醒 (Alerts)** - 持续显示直到关闭 ⭐ 推荐（不会错过）
- **横幅 (Banners)** - 5-10秒后自动消失（可能错过）
- **无** - 只在通知中心（❌ 不推荐）

**注意事项:**
- 支持通知中心历史记录
- 支持横幅和提醒样式
- 支持声音和徽章

### Windows

**支持版本:** Windows 10+

**配置步骤:**
1. Windows 10/11 自动支持通知
2. 首次使用时会自动注册通知
3. 如需调整：
   - 打开"设置" > "系统" > "通知和操作"
   - 找到 Intento
   - 开启通知权限

**注意事项:**
- 使用 WinRT 原生通知
- 支持操作中心
- 支持自定义声音

### Linux

**支持版本:** 主流发行版

**前置要求:**
- 需要安装通知守护进程

**常见通知守护进程:**
- `notify-osd` (Ubuntu 默认)
- `dunst` (轻量级)
- `mako` (Wayland)

**安装示例 (Ubuntu/Debian):**
```bash
sudo apt install notify-osd
```

**检查是否运行:**
```bash
pgrep -a notify-osd  # 或 dunst, mako
```

**注意事项:**
- 不同发行版通知样式可能不同
- 某些桌面环境需要额外配置

## ⚠️ 故障排除

### 通知不出现

**可能原因 1: 系统权限未开启**
- 检查操作系统通知设置
- 确保 Intento 有通知权限

**可能原因 2: 通知守护进程未运行 (Linux)**
- 运行 `pgrep notify-osd` 检查
- 如未运行，启动或安装守护进程

**可能原因 3: Intento 通知功能被关闭**
- 打开设置检查"启用桌面通知"是否开启
- 点击"测试通知"验证

### 测试通知成功，但截止提醒不工作

**可能原因 1: 提醒功能未开启**
- 确保"提醒我的截止日期"已勾选

**可能原因 2: 任务没有截止时间**
- 只有设置了截止时间的任务才会提醒
- 检查任务是否有 deadline 字段

**可能原因 3: 提醒时间还未到**
- 检查任务的截止时间
- 确认是否在提前提醒的时间范围内

### 通知太频繁

**解决方案:**
- 减少提前提醒时间（如从1天改为1小时）
- 只为重要任务设置截止时间
- 临时关闭提醒功能

### 通知太少

**解决方案:**
- 增加提前提醒时间（如从15分钟改为1天）
- 为更多任务设置截止时间
- 确保通知功能已开启

## 🔄 自动保存

### 保存行为

- **即时保存**: 所有设置更改立即保存
- **无需确认**: 不需要点击"保存"按钮
- **持久化**: 设置在应用重启后保持

### 保存位置

- 设置保存在应用数据目录
- 不同操作系统路径不同：
  - macOS: `~/Library/Application Support/com.intento.app/`
  - Windows: `%APPDATA%\com.intento.app\`
  - Linux: `~/.config/intento/`
