# Intento 用户指南

欢迎使用 **Intento (无忧记)** - 一款智能 Todo 桌面应用，通过 AI 帮助你快速管理任务，自动生成总结。

## 快速开始

### 核心理念：Command Palette

Intento 的核心是 **Command Palette** (⌘K / Ctrl+K)。把它想象成你的通用搜索和操作中心。

#### ⌘K 能做什么？
- 搜索所有任务
- 运行快捷操作
- 导航到应用的任何位置
- 筛选和查找任务

**专业提示**: 你可以完全用键盘操作所有功能！

---

## 键盘快捷键参考

### 全局快捷键

| 快捷键 | 操作 | 说明 |
|--------|------|------|
| `⌘K` / `Ctrl+K` | Command Palette | 打开通用命令中心 |
| `⌘N` / `Ctrl+N` | 新建任务 | 创建新任务（打开详情面板） |
| `⌘/` / `Ctrl+/` | AI 添加任务 | 打开自然语言任务输入 |
| `⌘R` / `Ctrl+R` | 查看总结 | 打开总结面板 |
| `⌘,` / `Ctrl+,` | 设置 | 打开应用设置 |
| `ESC` | 关闭 | 关闭任何打开的面板或对话框 |

### Command Palette 内快捷键

| 按键 | 操作 |
|------|------|
| `↑` `↓` | 上下导航 |
| `Enter` | 选择命令或任务 |
| `ESC` | 关闭面板 |
| 输入... | 过滤结果 |

### 任务卡片快捷操作

悬停在任何任务卡片上可显示快捷操作：

| 图标 | 操作 | 功能 |
|------|------|------|
| ✓ | 标记完成/待办 | 切换任务完成状态 |
| ✏️ | 编辑 | 在详情面板打开任务 |
| 🗑️ | 删除 | 移除任务（需确认） |

## 核心功能

### 1. 创建任务

#### 方式一：手动输入 (⌘N)
1. 按 `⌘N` 或点击"New Task"按钮
2. 在侧边面板填写详情
3. 点击"保存"

#### 方式二：AI 文本输入 (⌘/)
1. 按 `⌘/` 或点击"AI Add"按钮
2. 用自然语言描述任务
3. AI 自动提取标题、描述、优先级、截止时间和标签
4. 确认后自动创建任务

**AI 输入示例:**
- "周五前完成季度报告，高优先级"
- "明天买菜，低优先级，个人"
- "审查 PR #123 的代码，tag:work tag:urgent"

#### 方式三：图片识别
1. 按 `⌘/` 打开 AI 输入框
2. 粘贴图片 (`⌘V`) 或拖拽图片到输入框
3. AI 识别图片中的任务信息
4. 确认后创建任务

详见：[图片识别功能详解](./features/image-parsing.md)

### 2. 查找任务

#### 方式一：Command Palette (⌘K)
1. 按 `⌘K`
2. 输入搜索关键词
3. 结果显示匹配的任务
4. 用方向键导航
5. 按 `Enter` 打开任务

#### 方式二：视觉浏览
- 任务按优先级和状态自动排序
- 高优先级任务排在前面
- 彩色状态徽章
- 截止日期一目了然

### 3. 管理任务

#### 快速切换状态
1. 悬停在任务卡片上
2. 点击 ✓ 按钮
3. 状态在完成和待办之间切换

#### 完整编辑
1. 悬停并点击 ✏️，或
2. 点击卡片的任何位置
3. 在侧边面板编辑
4. 点击"保存"或按 `ESC` 取消

#### 删除任务
1. 悬停在任务上
2. 点击 🗑️ 按钮
3. 确认删除

### 4. 日历视图

切换到日历视图查看任务时间分布：

- **月视图**: 查看整月任务分布
- **任务可视化**: 彩色点指示优先级
- **点击日期**: 展开当日任务列表
- **筛选兼容**: 支持按标签过滤

### 5. 高级筛选

使用高级筛选面板精确查找任务：

- 按状态筛选 (Todo/Doing/Done)
- 按优先级筛选 (High/Medium/Low)
- 按标签筛选
- 按截止日期范围筛选
- 组合多个筛选条件

### 6. 统计面板

访问方式: `⌘K` → "View Statistics"

**显示内容:**
1. **概览卡片**
   - 总任务数
   - 待办数量
   - 进行中数量
   - 已完成数量

2. **完成率**
   - 可视化进度条
   - 完成百分比

3. **优先级分布**
   - 高、中、低优先级计数
   - 彩色编码快速扫描

4. **近期活动**
   - 最近 7 天完成的任务
   - 快速生产力检查

### 7. 自动总结

Intento 可以自动生成每日、每周、每月总结：

- **每日总结**: 每天固定时间自动生成
- **周总结**: 每周回顾
- **月总结**: 月度成果回顾
- **半年度/年度总结**: 长期成长轨迹

访问总结: 按 `⌘R` 或 `⌘K` → "查看总结"

详见：[总结功能使用指南](./features/summaries.md)

### 8. 通知系统

配置智能提醒，永不错过截止日期：

- **桌面通知**: 系统级通知提醒
- **截止日期提醒**: 在截止前指定时间提醒
- **灵活时间**: 可选 15分钟、30分钟、1小时、2小时、1天前提醒
- **测试功能**: 发送测试通知验证设置

详见：[通知设置指南](./features/notifications.md)

## 任务卡片详解

每个任务卡片显示：

```
┌─────────────────────────────────────────────┐
│ 任务标题                         [优先级]   │ ← 优先级徽章（仅高优先级）
│ 描述文本在这里...                           │ ← 描述第一行
│ [状态] 📅 截止时间  #标签 #标签            │ ← 元数据行
└─────────────────────────────────────────────┘
```

### 状态徽章
- **Todo** - 灰色徽章，任务未开始
- **Doing** - 蓝色徽章，正在进行
- **Done** - 绿色徽章，已完成（半透明显示）

### 优先级徽章
- **High** - 红色徽章（始终显示）
- **Medium** - 无徽章（默认）
- **Low** - 无徽章

### 截止日期显示
- **逾期** - 红色文字
- **今天** - 显示"今天"
- **明天** - 显示"明天"
- **7天内** - 显示"还有X天"
- **更晚** - 显示日期（如"12月15日"）

## 设置面板

访问方式: `⌘,` 或 `⌘K` → "设置"

### 通知设置
- **启用桌面通知**: 开关系统通知
- **提醒截止日期**: 在到期前获得提醒
- **提前提醒时间**: 提前多久提醒（默认：24小时）

### 自动总结设置
- **启用自动总结**: 开关自动总结功能
- **每日总结**: 设置生成时间
- **每周总结**: 选择生成日期
- **每月总结**: 月末自动生成

### AI 提供商设置
- 选择 AI 提供商（OpenAI/Claude/Kimi）
- 配置 API Key

### 外观设置
- **主题**: 当前为温暖浅色（深色模式即将推出）
- **紧凑模式**: 更紧凑的间距（未来功能）

## 常见问题

**Q: 如何搜索特定任务？**
A: 按 `⌘K` 并开始输入。匹配的任务会显示在命令下方。

**Q: 可以不打开任务就标记为完成吗？**
A: 可以！悬停在任务卡片上并点击出现的 ✓ 按钮。

**Q: 创建任务最快的方法是什么？**
A: 按 `⌘/` 并自然描述。例如："明天下午2点团队会议，高优先级"

**Q: 如何查看生产力统计？**
A: 按 `⌘K`，输入"statistics"，按 `Enter`。或点击 command palette 中的图表图标。

**Q: 可以不用鼠标使用 Intento 吗？**
A: 当然！这正是设计初衷。用 ⌘K 导航，方向键移动，Enter 选择。

**Q: 侧边栏去哪了？**
A: 我们移除了它以给你更多任务空间。所有功能现在都在 command palette (⌘K) 中。

**Q: 如何更改设置？**
A: 按 `⌘,` 或打开 command palette (⌘K) 搜索"settings"。

**Q: 已完成的任务会怎样？**
A: 它们保留在列表中，但以较低透明度和绿色勾选显示。你仍可以编辑或删除它们。

**Q: 可以测试通知吗？**
A: 可以！打开 command palette (⌘K) 并运行"Test Notification"。

## 故障排除

**Command palette 不打开？**
- 尝试 `Ctrl+K` (Windows/Linux) 或 `⌘K` (Mac)
- 确保没有其他应用捕获了该快捷键

**快捷键不工作？**
- 点击应用窗口使其获得焦点
- 检查是否有其他应用捕获了该快捷键

**任务无法保存？**
- 确保标题已填写（必填字段）
- 检查顶部的错误消息

**AI 解析不工作？**
- 确保有网络连接
- 尝试重新表述输入
- 回退到手动输入 (⌘N)

**通知不出现？**
- 检查系统通知权限
- 验证通知守护进程正在运行（Linux）
- 使用"测试通知"按钮测试
