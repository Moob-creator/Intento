# Intento 图片识别多操作功能 - 使用指南

## 🎯 功能概述

Intento 现在支持通过图片识别多种任务操作，包括创建、更新、完成、删除任务等。

## 📸 如何使用

### 方式 1: 键盘快捷键

1. 按 `⌘/` (macOS) 或 `Ctrl+/` (Windows/Linux) 打开 AI 输入框
2. 按 `⌘V` 粘贴图片或直接拖拽图片到输入框
3. 点击发送按钮或按 `⌘Enter`
4. 查看识别结果并选择要执行的操作
5. 点击"确认执行"

### 方式 2: 点击按钮

1. 点击顶部的"AI Add"按钮（带闪光图标）
2. 粘贴图片或拖拽图片到输入框
3. 点击发送按钮
4. 查看识别结果并选择要执行的操作
5. 点击"确认执行"

## 🎨 支持的操作类型

### 1. 创建任务 (Create)

**图片内容示例**:
```
明天下午 3 点前完成季度报告
优先级: 高
标签: 工作, 报告
```

**识别结果**:
- 操作类型: 创建任务
- 标题: 完成季度报告
- 截止时间: 明天 15:00
- 优先级: 高
- 标签: 工作, 报告

### 2. 更新任务 (Update)

**图片内容示例**:
```
将"季度报告"的截止时间改为周五
优先级改为中等
```

**识别结果**:
- 操作类型: 更新任务
- 任务: 季度报告
- 新截止时间: 周五
- 新优先级: 中等

### 3. 完成任务 (Complete)

**图片内容示例**:
```
已完成季度报告
```

**识别结果**:
- 操作类型: 完成任务
- 任务: 季度报告

### 4. 删除任务 (Delete)

**图片内容示例**:
```
删除"测试任务"
```

**识别结果**:
- 操作类型: 删除任务
- 任务: 测试任务

### 5. 批量完成 (BatchComplete)

**图片内容示例**:
```
完成以下任务:
- 写周报
- 发送邮件
- 更新文档
```

**识别结果**:
- 操作类型: 批量完成
- 任务列表: 写周报, 发送邮件, 更新文档

### 6. 设置状态 (SetStatus)

**图片内容示例**:
```
将"季度报告"状态设为进行中
```

**识别结果**:
- 操作类型: 设置状态
- 任务: 季度报告
- 新状态: 进行中 (doing)

## 🔍 确认对话框说明

### 对话框显示内容

1. **头部信息**
   - 识别到的操作总数
   - 图片描述（如有）
   - 识别置信度（百分比）
   - 警告信息（如有）

2. **操作列表**
   - 每个操作有图标和颜色标识
   - 显示操作类型和详细内容
   - 可以通过复选框选择要执行的操作
   - 默认全选

3. **底部操作**
   - 显示已选操作数
   - "取消"按钮 - 关闭对话框，不执行任何操作
   - "确认执行"按钮 - 执行选中的操作

## 💡 使用技巧

### 1. 图片质量

- ✅ 使用清晰的截图
- ✅ 确保文字清晰可读
- ✅ 避免反光或模糊

### 2. 描述方式

- ✅ 使用清晰的动词（创建、更新、完成、删除）
- ✅ 任务标题要明确
- ✅ 可以包含截止时间、优先级等信息

### 3. 任务标识

- 对于更新/完成/删除操作，使用任务的完整或部分标题
- 系统会使用模糊匹配查找任务
- 如果找不到匹配的任务，会显示错误信息

### 4. 批量操作

- 可以在一张图片中包含多个操作
- 系统会识别所有操作并让你选择
- 可以取消不需要的操作

## ⚠️ 注意事项

### 任务匹配

- 更新/完成/删除操作需要能找到对应的任务
- 如果标识符不清晰，可能找不到任务
- 建议使用完整的任务标题

### 错误处理

- 如果部分操作失败，成功的操作仍会执行
- 错误信息会显示具体的失败原因
- 可以关闭对话框后重试失败的操作

### 操作顺序

- 操作按顺序执行（不是并行）
- 如果某个操作失败，不会影响其他操作

## 🔄 工作流程示例

### 场景 1: 从会议纪要创建多个任务

1. 截取会议纪要中的待办事项列表
2. 粘贴图片到 Intento
3. 系统识别出多个"创建任务"操作
4. 查看识别结果，确认所有任务
5. 点击"确认执行"
6. 所有任务自动创建完成

### 场景 2: 从邮件更新任务状态

1. 截取邮件中关于任务状态的内容
2. 粘贴图片到 Intento
3. 系统识别出"更新任务"或"完成任务"操作
4. 确认操作正确
5. 点击"确认执行"
6. 任务状态自动更新

### 场景 3: 批量完成任务

1. 截取已完成任务的列表
2. 粘贴图片到 Intento
3. 系统识别出"批量完成"或多个"完成任务"操作
4. 取消不需要完成的任务
5. 点击"确认执行"
6. 选中的任务标记为完成

## 🎬 快速开始

1. **第一次使用**
   ```
   1. 打开 Intento
   2. 按 ⌘/ 打开 AI 输入
   3. 粘贴一张包含任务的图片
   4. 点击发送
   5. 查看识别结果
   6. 点击确认执行
   ```

2. **配置 API Key**
   ```
   确保已设置以下环境变量之一:
   - MOONSHOT_API_KEY (Kimi/Moonshot AI)
   - OPENAI_API_KEY (OpenAI)
   ```

3. **测试功能**
   ```
   创建一个测试图片:
   - 写上"明天完成测试任务"
   - 截图或拍照
   - 粘贴到 Intento
   - 查看识别结果
   ```

## 📊 性能说明

- **识别速度**: 2-8 秒（取决于网络和图片大小）
- **准确率**: >90%（清晰图片）
- **Token 消耗**: 约 1300-1400 tokens/图片
- **支持格式**: PNG, JPG, JPEG, WebP

## 🆘 常见问题

### Q: 识别结果不准确怎么办？
A:
- 确保图片清晰
- 使用明确的任务描述
- 检查是否包含足够的上下文信息

### Q: 找不到任务怎么办？
A:
- 确认任务标题正确
- 使用完整的任务名称
- 检查任务是否真的存在

### Q: 可以同时创建和更新任务吗？
A: 可以！系统支持在一张图片中识别多种不同类型的操作。

### Q: 支持哪些语言？
A: 主要支持中文和英文，其他语言的识别准确率可能较低。
