# Intento 产品路线图

本文档记录 Intento 的长期规划、未来功能和版本计划。

**最后更新:** 2026-02-19

---

## 🎯 产品愿景

打造一款"无感知"的任务管理工具，让用户专注于完成工作，而非管理任务。通过 AI 和自动化，最小化用户操作，最大化生产力。

---

## 📅 版本规划

### v0.1.0 - MVP 版本 ✅ (当前)

**发布时间:** 2026-02

**核心功能:**
- ✅ Command Palette 驱动界面
- ✅ 基础任务 CRUD
- ✅ AI 文本解析
- ✅ 图片识别（多操作）
- ✅ 智能提醒系统
- ✅ 自动总结生成（日/周/月）
- ✅ 日历视图（月视图）
- ✅ 高级筛选

**技术亮点:**
- Rust + Tauri 2.0 高性能架构
- React 19 + TypeScript 类型安全
- ADK-Rust AI 集成
- SQLite 本地数据存储

## 💡 功能提议流程

有新功能想法？欢迎提议！

### 提议步骤

1. **创建 Issue**
   - 使用 `[Feature Request]` 标签
   - 描述功能场景和价值

2. **社区讨论**
   - 其他用户反馈
   - 可行性评估

3. **优先级评定**
   - 团队评估优先级
   - 加入对应版本计划

4. **开发排期**
   - 分配到具体版本
   - 开始开发

### 评估标准

- **用户价值**: 解决真实痛点
- **实现成本**: 开发时间和复杂度
- **维护成本**: 长期维护难度
- **与愿景契合度**: 符合产品定位
